# ERP Health Monitor

ERP Health Monitor is a powerful Odoo utility module that helps administrators and developers monitor the overall health, performance, and stability of their Odoo ERP system.

It provides real-time insights into cron jobs, slow SQL queries, system logs, database locks, and server resource usage such as CPU and memory.
